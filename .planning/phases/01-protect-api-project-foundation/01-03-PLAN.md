    - loadCredentials() -> returns (host, username, password) tuple or null if any key missing
    - clearCredentials() -> deletes all 3 keys + selected_camera_ids + ssl_accepted

    - login(host, user, pass) success -> saves credentials, returns AuthState.authenticated
    - login(host, user, pass) failure -> returns AuthState.unauthenticated with error

     - `Future<void> saveCredentials(String host, String username, String password)` -- writes keys `protect_host`, `protect_username`, `protect_password`

     - `Future<void> saveSslAccepted(String host, bool accepted)` -- writes `accepted.toString()` to key `ssl_accepted_$host`

    **Imports:**
    ```dart

    import 'package:media_kit/media_kit.dart';

    ```

    **Provider declaration:**
    ```dart
    final audioPlayerProvider =
        AsyncNotifierProvider<AudioPlayerNotifier, MonitoringState>(
            AudioPlayerNotifier.new);
    ```

    **Class structure:**
    - Private field `final Map<String, Player> _players = {};` mapping cameraId to Player instance.
    - `build()` returns `MonitoringState()` (idle, empty cameras list).

    **`startMonitoring()` method:**
    1. Read `authNotifierProvider` to get `host` (the NVR IP).
    2. Read `cameraNotifierProvider` to get `selectedCameras`.
    3. Set state to `AsyncLoading()`.
    4. For each selected camera:
       a. Create `CameraAudioState(cameraId: camera.id, cameraName: camera.name ?? 'Camera', connectionStatus: CameraConnectionStatus.connecting)`.
       b. Create a `Player` with `PlayerConfiguration(protocolWhitelist: ['udp', 'rtp', 'tcp', 'tls', 'data', 'file', 'http', 'https', 'crypto', 'rtsp', 'rtsps'], bufferSize: 2 * 1024 * 1024)`.
       c. Get `NativePlayer` via `player.platform as NativePlayer`.
       d. Set `await nativePlayer.setProperty('vid', 'no')` -- disable video decoding (STRM-01).
       e. Set `await nativePlayer.setProperty('profile', 'low-latency')`.
       f. Set `await nativePlayer.setProperty('cache', 'no')`.
       g. Set `await nativePlayer.setProperty('demuxer-lavf-o', 'rtsp_transport=tcp')`.
       h. Store in `_players[camera.id] = player`.
       i. Open stream: `await player.open(Media(rtspUrl(host!, camera.id)))`.
       j. On success, update camera state to `connectionStatus: CameraConnectionStatus.playing`.
       k. Wrap in try/catch -- on error, set `connectionStatus: CameraConnectionStatus.error, errorMessage: e.toString()`.
    5. Emit `AsyncData(MonitoringState(cameras: [cameraStates]))`.
    6. Log via `appLog('AUDIO', ...)` at each step.

    **`setVolume(int cameraIndex, double volume)` method:**
    1. Get current state cameras list.
    2. Get the CameraAudioState at `cameraIndex`.
    3. Call `_players[cameraState.cameraId]!.setVolume(cameraState.isMuted ? 0.0 : volume)`.
    4. Update state via `copyWithCamera(cameraIndex, cameraState.copyWith(volume: volume, preMuteVolume: volume))`.

    **`setPan(int cameraIndex, double pan)` method:**
    1. Get the CameraAudioState at `cameraIndex`.
    2. Get `NativePlayer` via `_players[cameraState.cameraId]!.platform as NativePlayer`.
    3. Call `await nativePlayer.setProperty('af', buildPanFilter(pan))`.
    4. Update state via `copyWithCamera(cameraIndex, cameraState.copyWith(pan: pan))`.

    **`toggleMute(int cameraIndex)` method:**
    1. Get the CameraAudioState at `cameraIndex`.
    2. If currently muted: set `isMuted: false`, call `player.setVolume(cameraState.preMuteVolume)`.
    3. If not muted: store current volume as `preMuteVolume`, set `isMuted: true`, call `player.setVolume(0.0)`.

    **`stopMonitoring()` method:**
    1. For each player in `_players.values`: call `await player.stop()` then `await player.dispose()`.
    2. Clear `_players`.
    3. Set state to `AsyncData(MonitoringState())`.
    4. Log `appLog('AUDIO', 'All players stopped and disposed')`.

    **Disposal via `ref.onDispose()`:**
    Register in `build()`: `ref.onDispose(() { for (final p in _players.values) { p.dispose(); } _players.clear(); })`.

    **Tests** (`test/features/monitoring/audio_player_provider_test.dart`):
    Since Player requires native libraries, tests focus on state logic:
    - Test that MonitoringState transitions are correct (create state, apply setVolume, verify effectiveVolume)
    - Test toggleMute logic: muted state has effectiveVolume 0.0, unmuted restores preMuteVolume
    - Test setPan updates pan value in state
    - These are pure state tests using CameraAudioState/MonitoringState directly (no Player mocking needed)

    ```dart
    class MonitoringScreen extends ConsumerStatefulWidget {
      @override
      ConsumerState<MonitoringScreen> createState() => _MonitoringScreenState();
    }

    class _MonitoringScreenState extends ConsumerState<MonitoringScreen> {
      @override
      void initState() {
        super.initState();
        // Start monitoring when screen loads
        Future.microtask(() {
          ref.read(audioPlayerProvider.notifier).startMonitoring();
        });
      }

      @override
      Widget build(BuildContext context) {
        final monitoringState = ref.watch(audioPlayerProvider);

        return Scaffold(
          appBar: AppBar(title: const Text('Monitoring')),
          body: monitoringState.when(
            loading: () => const Center(child: CircularProgressIndicator()),
            error: (e, _) => Center(child: Text('Error: $e')),
            data: (state) {
              if (state.cameras.isEmpty) {
                return const Center(child: CircularProgressIndicator());
              }
              return Padding(
                padding: const EdgeInsets.all(Spacing.lg),
                child: Column(
                  children: [
                    for (int i = 0; i < state.cameras.length; i++) ...[
                      CameraAudioCard(
                        cameraState: state.cameras[i],
                        cameraIndex: i,
                      ),
                      if (i < state.cameras.length - 1)
                        const SizedBox(height: Spacing.lg),
                    ],
                  ],
                ),
              );
            },
          ),
          bottomNavigationBar: const StopMonitoringButton(),
        );
      }
    }
    ```
